# Coronavirus

## Feature
- Lists weekly new covid cases.
- Can change area which your want to know.
- Uses open data API from GOV.UK.
- Supports dynamic color.

## Architecture
- Presentation
  - View
  - ViewModel
- Data
  - Repository
  - DataSource

## Library
  - ViewBinding
  - LiveData
  - Coroutine
  - Retrofit, Gson
  - Junit
